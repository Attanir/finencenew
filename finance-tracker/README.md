# Мои финансы — приложение для Android

Простое приложение для учёта расходов и баланса кошельков с пересчётом в MYR по актуальному курсу.

## Что умеет (v1.1)
- Операции двух типов: **расход** и **приход** (зарплата, возврат, подработка, подарок, перевод и т.д.).
- У каждой операции указывается **кошелёк-источник** — баланс этого кошелька автоматически уменьшается (расход) или увеличивается (приход).
- Категории расходов: жильё, еда, рестораны, транспорт, кредиты, одежда, игрушки, развлечения.
- Баланс кошельков (Kaspi, Alliance, наличные, GrabPay, Touch'n Go) — стартовые суммы заполняете вручную, дальше они меняются операциями (и всегда редактируются вручную).
- Суммирование всех кошельков с переводом в MYR по текущему курсу (KZT, USD и др.).
- Сводка за месяц: приход / расход + разбивка расходов по категориям.
- **Экспорт в Instagram Stories** — кнопка «Поделиться в Stories» делает картинку 1080×1920 с итогом и сохраняет её в Галерею + открывает меню «Поделиться».
- Курсы валют тянутся бесплатно с open.er-api.com (ExchangeRate-API), кэшируются для работы офлайн.

## Как собрать APK через GitHub (без программирования)

1. Зарегистрируйтесь на https://github.com (если аккаунта ещё нет).
2. Нажмите **New** → создайте новый репозиторий (имя любое, например `moi-finansy`), тип **Public**, без галочек «Add README». Нажмите **Create repository**.
3. На странице репозитория нажмите **uploading an existing file** (ссылка в подсказке) или **Add file → Upload files**.
4. Перетащите в окно **всё содержимое** этой папки (включая папки `app`, `gradle`, `.github` и файлы `gradlew`, `settings.gradle` и т.д.). Перетаскивание папок через браузер сохраняет структуру.
5. Внизу нажмите **Commit changes**.
6. Откройте вкладку **Actions** вверху. Сборка запустится сама. Подождите 2–4 минуты до зелёной галочки ✓.
7. Кликните на завершившийся запуск → внизу в разделе **Artifacts** скачайте **MoiFinansy-APK** (это zip).
8. Распакуйте zip — внутри `app-debug.apk`. Перекиньте его на телефон.
9. На Android при установке разрешите «установку из неизвестных источников» для браузера/проводника. Установите APK.

Готово. Приложение работает офлайн; для обновления курсов нужен интернет.

## Заметки
- Валюты кошельков можно менять. По умолчанию: Kaspi — KZT, остальные — MYR.
- Все суммы вводите вручную.
- Валюта отображения по умолчанию MYR, меняется на экране «Баланс».
- Курсы: ExchangeRate-API · open.er-api.com
