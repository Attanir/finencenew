# Финансы — Android APK

Приложение для учёта кошельков и расходов (то же, что веб-версия), упакованное в Android.
Сборка APK происходит автоматически на серверах GitHub — ничего ставить на компьютер не нужно.

## Как получить APK

1. Создай новый репозиторий на GitHub (Public или Private — без разницы).
2. Загрузи в него **всё содержимое этой папки**, включая скрытую папку `.github`
   (Add file → Upload files → перетащить файлы → Commit).
3. Открой вкладку **Actions** — сборка «Build APK» запустится сама.
   Подожди ~3–5 минут до зелёной галочки.
4. Зайди в завершившийся запуск → блок **Artifacts** внизу → скачай **finance-apk**.
5. Распакуй архив — внутри `app-debug.apk`.

## Как установить на телефон

1. Перекинь `app-debug.apk` на телефон (через кабель, Telegram себе, Google Drive и т.п.).
2. Открой файл на телефоне. Android попросит разрешить установку из этого источника —
   разреши (Настройки → «Установка неизвестных приложений»).
3. Готово, иконка появится на экране.

Это debug-сборка: для личного использования и установки вручную подходит полностью.
Для публикации в Google Play нужна отдельная подпись — это уже другая история.

## Что внутри

- `app/src/main/assets/finance.html` — само приложение. Хочешь обновить —
  просто замени этот файл и снова запусти сборку (вкладка Actions → Run workflow).
- `.github/workflows/build.yml` — сборщик APK.
